# MVC_CV
**MVC ile Dinamik Cv Sitesi** 
Bu proje, kullanıcıların özgeçmişlerini oluşturabileceği ve yönetebileceği dinamik bir CV projesidir.
#### **`Kullanılan Teknolojiler`**
+ C#
+ HTML
+ CSS
+ JS
+ ASP.NET Framework
+ MVC
+ Entity Framework
+ Linq
+ MsSQL Server
+ Repository Design Pattern
